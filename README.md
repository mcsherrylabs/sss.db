# sss.db
Scala module for basic sql db access.

[![Build Status](https://travis-ci.org/mcsherrylabs/sss.db.svg?branch=master)](https://travis-ci.org/mcsherrylabs/sss.db) [![Coverage Status](https://coveralls.io/repos/mcsherrylabs/sss.ancillary/badge.svg?branch=master&service=github)](https://coveralls.io/github/mcsherrylabs/sss.ancillary?branch=master)

A simple way to deal with reading and inserting into sql tables. 
Supports transactions and optimistic versioning.
